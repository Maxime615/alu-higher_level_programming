# Python - Data Structures: Lists, Tuples

This project focuses on understanding and working with fundamental data structures in Python, specifically lists and tuples.

## Tasks Summary

| File | Description |
| --- | --- |
| `1-element_at.py` | Retrieves an element from a list like in C. |
| `2-replace_in_list.py` | Replaces an element of a list at a specific position. |
| `3-print_reversed_list_integer.py` | Prints all integers of a list in reverse order using `str.format()`. |
| `4-new_in_list.py` | Replaces an element in a list at a specific position without modifying the original. |
| `5-no_c.py` | Removes all characters `c` and `C` from a string without using `str.replace()`. |
| `6-print_matrix_integer.py` | Prints a matrix of integers cleanly with exact spacing. |
| `7-add_tuple.py` | Adds two tuples together up to their first 2 elements. |
| `8-multiple_returns.py` | Returns a tuple with the length of a string and its first character. |
| `9-max_integer.py` | Finds the biggest integer of a list without using the built-in `max()`. |
| `10-divisible_by_2.py` | Finds all multiples of 2 in a list and returns a list of booleans. |
| `11-delete_at.py` | Deletes the item at a specific position in a list without using `pop()`. |
| `12-switch.py` | A 5-line script that switches the values of two variables. |
